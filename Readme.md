# ♥ For Hager

A romantic love website dedicated to Hager — built with love, animated with passion.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Motion (animations)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

Output will be in the `dist/` folder.

## Customize

Edit `src/lib/constants.ts` to:
- Add your own photos (replace placeholders in `GALLERY_IMAGES`)
- Write your love story (`STORY_MILESTONES`)
- List your reasons (`REASONS`)
- Write your letter (`LETTER_TEXT`)
